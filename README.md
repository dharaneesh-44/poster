# 🎨 Social Media Poster Design Collection

A creative collection of **social media promotional posters** designed in **Figma**. This project showcases modern marketing creatives with clean layouts, bold typography, attractive color schemes, and high-quality product presentations suitable for advertising campaigns and branding.

---

## ✨ Project Overview

This design collection includes promotional posters created for different products and marketing campaigns. Each poster is designed to grab attention while maintaining a professional and visually appealing style.
